about=Lets you pick a PostgreSQL connection configured in QGIS and run a dump (via ogr2ogr) to a GeoPackage file, choosing whether to export the whole database or a single schema. Useful for large databases where you don't want to load every layer into QGIS. Requires the "ogr2ogr" command (part of GDAL, bundled with standard QGIS installations) to be available in QGIS' PATH, and the "psycopg2" Python library in QGIS' Python environment (usually already present; if not, install it with "pip install psycopg2" in the QGIS Python environment, or on Linux with "sudo apt install python3-psycopg2"). Windows/OSGeo4W installations already ship both.
